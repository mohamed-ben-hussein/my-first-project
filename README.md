اسم المشروع ( Task Manger)

قمت ببناء هذا المشروع لأتدرب و اطبق عليه كل ما اتعلمه في بايثون و البرمجة بصفة عامة و سيتطور كلما يتطور مستواي في البرمجة 

🚀 ميزات المشروع
الميزة الأولى ( يمكن للمستخدم اضافة مهمة ) هناك مزيد من الميزات قادمة 

🛠️ التقنيات المستخدمة

اللغة: Python

أدوات أخرى: Git, GitHub

📋 المتطلبات الأساسية (Prerequisites)
قبل تشغيل المشروع، تأكد من تثبيت: Python 3.14.4

# استنساخ المشروع : git clone https://github.com/mohamed-ben-hussein/my-first-project.git 
# الدخول لمجلد المشروع: cd my-first-project
# تثبيت المكتبات : install npm
# تشغيل المشروع : npm start
